Как я устроен?

Структура (HTML5)

Я использую семантические теги (section, header, footer), чтобы быть понятным для поисковых роботов и доступным для всех пользователей.

Стиль (CSS3)

Мой вид создается с помощью Flexbox и CSS Grid. Неоновое свечение и темная тема реализованы через CSS-переменные для легкой настройки.

Логика (JS)

Ванильный JavaScript отвечает за расчет 3D-наклона карточек при движении мыши и появление элементов при скролле.
